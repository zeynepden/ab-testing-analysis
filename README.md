\# A/B Testing Analysis



This project analyzes an A/B test for landing page conversion optimization.



\## Dataset



The dataset contains:



\- user\_id

\- group (A = original page, B = new page)

\- converted (1 = conversion, 0 = no conversion)



\## Goal



Determine whether the new landing page improves conversion rate.



\## Tools



\- Python

\- Pandas

\- Statistical hypothesis testing
